GeoAI: Assisted OSM Explorer

About this project

Minimal AI-assisted Point of Interest (POI) Explorer that interacts with OpenStreetMap (OSM) data.

The system utilizes a Large Language Model (LLM) to:

  • Interpret user requests submitted in natural language.
  • Automatically generate the necessary geospatial queries using the Overpass Query Language (OQL) format.
  • Dynamically visualize the results on a map, using coherent categories for the POIs such as restaurant, school, park, etc.
